Chrome Flatpak Beta

Please be aware of the following as you use this:

  • Do NOT immediately file bugs on the Chromium bug tracker! File them to the corresponding Flathub tracker instead.
  • USB devices such as game controllers are not available inside the sandbox yet due to this issue. This also means any U2F keys likely will not work.
  • Installing PWAs / websites as apps silently will not work.
  • This Flatpak may break from time to time while still in beta.

Enjoy!
